# Algebra of selection
### Steps to run
1. Install rustup using winget: `winget install -e --id Rustlang.Rustup`
2. Install rust compiler and tools: `rustup toolchain install stable`
3. Clone this repository locally using: `git clone https://github.com/manan-gup/algebra-of-selection.git`
4. Get symbolica hobbyist license following the instructions from the [symbolica docs](https://symbolica.io/docs/get_started.html#non-professional-users)
5. Set the key obtained from the email in the `LicenseManager::set_license_key()` command at the top of the `src/main.rs` file
6. Run code by using `cargo run` within the project folder
